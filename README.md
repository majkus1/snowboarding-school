# Pow Pow Riders

Website for Pow Pow Riders snowboard school.

## About

Pow Pow Riders offers:
- SITS instructor training (Level II, III, Assistant Instructor)
- Individual and group lessons
- Camps and courses (weekend programs, glacier training)
- Mental training workshops

## Tech Stack

- Next.js 13 + React 18
- SCSS
- GSAP, React Scroll Parallax
- Bootstrap 5
- WordPress integration for store

## Structure

```
pages/          # Next.js pages
components/     # React components
styles/         # SCSS styles
public/         # Static assets
```
